# pedestrian-traffic-control-system-arduino
Arduino-based Pedestrian Traffic Control System that simulates a real pedestrian crossing using Red, Yellow and Green LEDs, a push button, and a 16x2 I2C LCD. The system safely manages traffic, counts pedestrians after each crossing request, and displays the total count, demonstrating embedded systems and automation concepts. using Arduino Uno. V1.
